Signature Verification

Each webhook includes an X-Webhook-Signature header with HMAC-SHA256 signature. Verify it using your webhook secret to ensure authenticity.

Example Node.js verification:

const crypto = require('crypto'); function verifyWebhook(payload, signature, secret) { const expectedSignature = crypto .createHmac('sha256', secret) .update(JSON.stringify(payload)) .digest('hex'); return signature === expectedSignature; } // In your webhook endpoint: const signature = req.headers['x-webhook-signature']; const isValid = verifyWebhook(req.body, signature, 'whsec_test_abc123'); if (!isValid) { return res.status(401).send('Invalid signature'); } // Process webhook res.status(200).send('OK');
